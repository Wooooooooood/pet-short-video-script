# 萌宠短视频脚本生成器

给自家毛孩子拍短视频，却不知道怎么写脚本？这个 Skill 帮你从零到一，把宠物的性格特点变成可直接拍摄的完整分镜脚本——附带 AI 视频工具（Seedance 2.0、Kling、Veo 等）能直接读懂的英文画面提示词。

## 它能做什么

- **聊几句就开工** — 不用填表格，像聊天一样说说你家宠物的品种、长相、性格，系统自动整理成角色档案
- **故事方向不用想** — 提供 10+ 个经过验证的萌宠故事模板（搞笑反差、温馨治愈、复仇小剧场、秘密生活……），还支持模板混搭，打造独一无二的剧情
- **每个镜头都写清楚** — 自动生成包含时间轴、景别、运镜、画面描述、台词/字幕、音效提示的完整分镜表
- **AI 视频工具直接能用** — 每个镜头附带 40-80 词的英文 Seedance Prompt，保持宠物外观全程一致，复制粘贴即可生成画面

## 适合谁用

- 想给宠物拍抖音/小红书/YouTube Shorts 的铲屎官
- 用 Seedance、Kling、Veo 等 AI 视频工具做萌宠内容的创作者
- 有宠物照片但不知道怎么变成短视频故事的初学者

## 怎么开始

直接说：**"帮我给我家 [宠物品种] 拍个短视频"**，系统会引导你一步步完善想法。如果你已经想清楚故事了，也可以一次性说完，直接生成分镜。

## 支持的平台与格式

| 平台 | 默认比例 | 支持时长 |
|------|---------|---------|
| 抖音 / TikTok / Reels | 9:16 竖屏 | 15s / 30s / 60s |
| YouTube Shorts | 9:16 竖屏 | 15s / 30s / 60s |
| 小红书 | 9:16 或 1:1 | 15s / 30s / 60s |

## 支持的宠物

猫、狗是最拿手的，兔子、仓鼠、鹦鹉等小众宠物也完全没问题，会自动调用对应的拍摄技巧（比如仓鼠用 macro 特写 + 超大比例道具制造萌感）。

## 常见问题速览

- **没有宠物照片？** 没关系，用文字详细描述外观就行，照片是可选的
- **不知道拍什么故事？** 说"随便"，系统会根据宠物性格自动推荐 2 个方向
- **生成的脚本不满意？** 可以要求调整单个镜头、修改故事走向、换模板、改时长，随时迭代
- **为什么提示词是英文？** AI 视频模型对英文的理解更精准，你的输入全程用中文即可
